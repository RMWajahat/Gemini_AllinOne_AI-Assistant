## ğŸš€ Getting Started

### Prerequisites

*   **Node.js** (v18 or higher)
*   **Gemini API Key:** Get your free key from [Google AI Studio](https://aistudio.google.com/app/apikey).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/RMWajahat/Gemini_AllinOne_AI-Assistant.git
    cd Gemini_AllinOne_AI-Assistant
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in browser:** Navigate to `http://localhost:5173`.
